In this README, the purpose of each folder is explained. For further details on each file in each folder, reading the README file in each folder is recommended.

-Data_Preperation: Includes all the files relating to merging the initial raw data, passage extraction, and question creation

-Labeling: Includes all the files related to the labeling process, either the data preparation either results of labeling

-Model: Includes all the files related to our models and their running process

-Output: The final output of our models
